The Opacity in Web-designing

The opacity property sets the opacity level for an element. The opacity level describes the elements opaque level,where one is opaque, 0.5 is 50% see-through, and 0 is completely transperent. Below are some examples of how different opacity values can be applied to content blocks.

opacity 0.1

This block is very transparent. It seems almost invisible

opacity 0.3

This block is less tranperent than the first one. But still its being quite light.

opacity 0.6

At this level of the opacity the content becomes much more visible

opacity 1 (default)

The block is fully opaque now with no tranperency.
